# 🚀 Atividades com Vetores e Laços de Repetição (JavaScript)

📌 Descrição
Este repositório contém uma série de exercícios desenvolvidos utilizando estruturas de repetição e manipulação de **Vetores (Arrays)** em JavaScript. O objetivo principal é praticar o armazenamento de dados em coleções, iteração de elementos, geração de dados aleatórios e filtragem de arrays.

🛠️ Ferramentas Utilizadas
* Flowgorithm
* Visual Studio Code
* Git e GitHub
* Navegador Web / Node.js
* JavaScript

📋 Exercícios

1️⃣ Soma de Números Pares
O programa lê 10 números inteiros fornecidos pelo usuário e os armazena em um vetor. Em seguida, o algoritmo percorre esse vetor, identifica quais números são pares e exibe a soma total apenas desses valores.

2️⃣ Gerador de Vetor Aleatório
O sistema preenche automaticamente um vetor numérico com 7 números gerados aleatoriamente pelo computador. Após o preenchimento, o programa percorre o vetor e mostra na tela todos os valores que foram gerados.

3️⃣ Contador de Ocorrências
O programa solicita 10 números inteiros ao usuário e os armazena em um vetor. Depois, pede para o usuário digitar um número extra e faz uma busca no vetor para verificar e exibir quantas vezes esse número específico aparece na lista.

4️⃣ Remoção de Valores Duplicados
O sistema pede para o usuário digitar 10 números e os armazena em um vetor. Na sequência, o programa analisa os dados, remove todos os valores repetidos e cria um novo vetor limpo. Ao final, são exibidos tanto o vetor original quanto o vetor sem duplicatas.

💡 Objetivo de Aprendizado
* Compreender o conceito de indexação e armazenamento de dados em Vetores (Arrays).
* Aplicar estruturas de repetição (`for`, `while` ou `do-while`) para percorrer e manipular coleções.
* Utilizar métodos de geração de números pseudoaleatórios (como `Math.random()`).
* Desenvolver lógica de busca, contagem e filtragem de elementos dentro de um array.
